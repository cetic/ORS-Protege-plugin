Protégé Code Generation Plugin

The purpose of this plugin is to ease the transition from ontology development to application development. It generates java code allowing application developers to access and modify an ontology. It is similar to the Protégé 3 code generation plugins except that the code generated by this plugin is used with the Manchester OWL API. The plugin provides access to the code generation capability through a code generation menu item under the tools menu.
